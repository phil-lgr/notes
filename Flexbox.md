# CSS3 Flexbox

## Defaults

```bash
# flex childs, the flex-basis property defaults to auto
flex-basis: auto

# notice that the short hand does not set the same flex-basis as the default
flex: 1; // flex-grow: 1; flex-shrink: 1; flex-basis: 0;
flex: 20px; // flex-grow: 1; flex-shrink: 1; flex-basis: 20px;
flex: 0 40px; // flex-grow: 0; flex-shrink: 1; flex-basis: 40px;
```
